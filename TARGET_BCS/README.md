## Data Availability
The original dataset and execution environment used for this analysis were part of a guided business case study and are not publicly accessible.

The analysis, queries, results, and insights presented here are based on the outputs provided within the case study material. This repository focuses on analytical reasoning, interpretation of results, and business insights derived from the data.

## Reproducibility Note
While the original data is not included, the SQL logic and analytical approach documented in this project can be applied to similar e-commerce datasets with customer, order, payment, and logistics tables.
