# EV Charging Infrastructure Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes the distribution of EV charging infrastructure across India.

The dashboard provides insights into:

- Total charging stations
- States covered
- Cities covered
- Dominant charger type
- Top states by charging stations
- Top cities by charging stations
- Geographic distribution of charging stations

---

## Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization

---

## Key Insights

- Maharashtra has the highest number of charging stations.
- Fast DC is the dominant charger type.
- Delhi, Bengaluru, Chennai, Mumbai, and Hyderabad are major EV charging hubs.
- Charging infrastructure is concentrated around major urban centers.
